# Benchmarks

## UX / visual benchmarks
Use benchmarks as directional inspiration, not as templates to copy.

### Direction A: Editorial luxury
- Strong serif headings
- Quiet whitespace
- Soft neutral palette
- Refined composition

### Direction B: Premium planning app
- Calm dashboard cards
- Clear tables
- Large cost numbers
- Minimal but useful filtering

### Direction C: Soft bridal experience
- Emotional warmth without kitsch
- Delicate accent colors
- Soft surfaces and elegant motion

## Benchmark reminders
- The app should feel like a luxury wedding planner notebook turned into software.
- It should not feel like a finance admin panel.
- It should not feel like a generic feminine template.
