# Features

## MVP

### 1. Dashboard
- Total budget
- Confirmed costs
- Remaining to pay
- Estimated final total
- Cost per guest
- Active scenario selector
- Upcoming payments
- Overdue tasks

### 2. Budget and cost tracking
- Categories
- Cost items
- Pricing model: fixed, per_guest, per_child, per_room, conditional, manual
- Source type: venue_offer, vendor_quote, manual_estimate, actual
- Status: estimated, quoted, booked, deposit_paid, paid, cancelled
- Attachments and notes

### 3. Venue presets
- Venue-specific package rules
- Included items vs optional items
- Ability to override quantities and values
- Scenario-specific venue configuration

### 4. Guest list
- Household view
- Individual guest view
- Adults, children, infants, plus-ones
- RSVP status
- Meal preferences
- Accommodation needs
- Relationship tags
- Seating notes

### 5. Vendor management
- Category
- Contact details
- Quote range
- Selected offer
- Contract status
- Files and notes
- Payment schedule

### 6. Payment timeline
- Deposits
- Due dates
- Amounts due
- Paid / unpaid / overdue
- Linked vendor or venue item

### 7. Checklist and planning calendar
- Milestones by wedding date
- Ceremony formalities
- Venue tasks
- Guest-related tasks
- Vendor booking tasks

### 8. Scenario comparison
- Save multiple scenarios
- Compare cost totals
- Compare per-guest economics
- Compare included options and overrides
- Duplicate scenario and iterate

## Later phases
- Seating planner
- Envelope forecast
- PDF exports
- Vendor recommendation AI
