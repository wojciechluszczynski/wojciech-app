# Prompt 02 — Design system

Read CLAUDE.md and DESIGN_DIRECTION.md and ui_tokens.json.

Create the visual foundation for the app.

Tasks:
1. Define design tokens in code.
2. Create typography system.
3. Create app shell layout.
4. Create reusable components for cards, tables, empty states, badges, forms, tabs, and scenario selectors.
5. Keep the style elegant, soft, bridal, editorial, and highly readable.

Important:
- Avoid generic SaaS dashboard aesthetics.
- Use serif for display moments and sans for UI clarity.
- Financial values must be extra legible.
- Build mobile-first.
